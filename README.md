# Catch As Cat Can

A small console game in C# in which you play a cat, doing cat things. Sneak past the terrible human! Steal the food!

=^..^= *you, the cat*

|FOOD| *the food, which you must steal*

|'<'| *the terrible human*

**Controls** <br>
UP: Move up <br>
DOWN: Move down <br>
LEFT: Move left <br>
RIGHT: Move right <br>
ESC: Exit the game
